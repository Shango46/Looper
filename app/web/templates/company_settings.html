Web Search (Brave Search API)

What is this and how do I get a key?

Agents use the Brave Search API to search the live web — finding current news, documentation, prices, or any real-time information not in the company knowledge base.

Free tier: 2,000 queries/month at no cost. No credit card required.

To get an API key:

  1. Go to api.search.brave.com and create a free account.
  2. Under Subscriptions, select the Free AI plan (2,000 req/month).
  3. Copy the API key shown and paste it below.

The key is stored encrypted. Agents call web_search(query) and results are injected directly into their context.

Provider setup tips & settings reference
Gmail (personal) & Google Workspace
  • Enable 2-Step Verification, then go to Google Account → Security → App passwords and generate a password for "Mail". Use that instead of your real password.
  • SMTP: smtp.gmail.com · port 587 · STARTTLS ✓
  • IMAP: imap.gmail.com · port 993 · SSL ✓
  • Username is your full Gmail address (you@gmail.com or you@yourdomain.com for Workspace).
Zoho Mail
  • Go to Zoho Mail → Settings → Security → App Passwords to generate a password for Looper.
  • SMTP: smtp.zoho.com · port 587 · STARTTLS ✓  (or port 465 · SSL, uncheck STARTTLS)
  • IMAP: imap.zoho.com · port 993 · SSL ✓
  • Free tier allows up to 5 users and includes SMTP/IMAP access.
Fastmail
  • Go to Settings → Privacy & Security → Third-party apps → New app password. Use that password here.
  • SMTP: smtp.fastmail.com · port 587 · STARTTLS ✓
  • IMAP: imap.fastmail.com · port 993 · SSL ✓
Outlook / Microsoft 365
  • Microsoft is moving away from plain-password SMTP — check that your tenant still allows SMTP AUTH (admin panel: Users → Mail → SMTP AUTH). Personal @outlook.com accounts still work.
  • SMTP: smtp.office365.com · port 587 · STARTTLS ✓
  • IMAP: outlook.office365.com · port 993 · SSL ✓
General tips
  • Always use an App Password (not your main account password) — it's scoped, revocable, and works even if you have 2FA.
  • Use the Test SMTP and Test IMAP buttons above before saving to catch misconfigured ports or wrong credentials early.
  • For best deliverability, make sure your sending domain has SPF, DKIM, and DMARC DNS records — without them, outgoing agent emails may land in spam.
  • SMTP username and IMAP username are usually the same full email address.
