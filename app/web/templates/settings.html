Remote Access (Android app)

When enabled, a "Remote Access" section appears in each company's settings where you can generate a pairing code for the Android app. Requires Tailscale to be installed and running — the Tailscale IP is used as the host address the Android app connects to. Disabled by default; the rest of Looper works fully without it.
